# Medrec

Decentralized Medical Record Management App

## How to Run Locally

### Software Requirements

- Node
- Ganache
- Truffle  
- Metamask Extension

### Start Local Blockchain

```
ganache
```

### Compile Contracts

```
truffle migrate 
cd utils 
node helper.js 
```

- Please verify each of deployed contract's address in Config/config.js is similar to that in build folder.

### Start Frontend

```
npm i
npm run dev
```
